# Study Room Reservation System

##  Project Description
This is a simple web-based prototype for reserving study rooms at UTSA. 
Users can search for available rooms, reserve a room, and cancel reservations. 
The system demonstrates core functionality using HTML, CSS, and JavaScript.

---

##  Features
- Search for available study rooms
- Reserve a study room
- Cancel a reservation
- Display confirmation messages
- Show error messages when no rooms are available
